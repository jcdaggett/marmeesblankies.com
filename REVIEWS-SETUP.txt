============================================================
  MARMEE'S BLANKETS - REVIEW SUBMISSION SETUP
============================================================

This lets customers scan a QR code, land on a review page, and
submit a review that gets stored for you to check and approve.

Submitted reviews do NOT go live automatically. You check them
in the Cloudflare KV dashboard and send the good ones to Claude
to add to the site (same as always).

You do this once.

------------------------------------------------------------
  STEP 1 - CREATE THE KV NAMESPACE (where reviews get stored)
------------------------------------------------------------
1. Go to https://dash.cloudflare.com
2. Left menu: Storage & Databases -> KV
3. Click "Create a namespace"
4. Name it:  marmees-reviews   -> Add

------------------------------------------------------------
  STEP 2 - CREATE THE WORKER
------------------------------------------------------------
1. Left menu: Workers & Pages -> Create -> "Create Worker"
2. Name it:  marmees-reviews   -> Deploy (makes a hello-world)
3. Click "Edit code"
4. Delete everything in the editor
5. Open the file  reviews-worker.js  (in this folder) in Notepad,
   copy ALL of it, paste into the Cloudflare editor
6. Click "Deploy"

------------------------------------------------------------
  STEP 3 - CONNECT THE KV NAMESPACE TO THE WORKER
------------------------------------------------------------
1. In the Worker page, go to:  Settings -> Bindings
2. Add binding -> KV Namespace
      Variable name:   REVIEWS
      KV namespace:    marmees-reviews (the one from Step 1)
3. Save / Deploy

------------------------------------------------------------
  STEP 4 - CONNECT THE REVIEW PAGE TO THE WORKER
------------------------------------------------------------
1. Your Worker has a URL like:
      https://marmees-reviews.jesse-daggett.workers.dev
   Copy it from the Worker's page.
2. Open  leave-a-review.html  in Notepad. Ctrl+F for:
      REVIEWS_WORKER_URL
   You'll find:   const REVIEWS_WORKER_URL = "";
3. Paste your Worker URL between the quotes.
4. Save. Double-click deploy.bat to push it live.

------------------------------------------------------------
  STEP 5 - MAKE THE QR CODE
------------------------------------------------------------
Point the QR code at:
      https://marmeesblankets.com/leave-a-review.html

Any free QR generator works (e.g. qr-code-generator.com).
Test it with your phone before printing stickers.

------------------------------------------------------------
  STEP 6 - CHECKING SUBMITTED REVIEWS
------------------------------------------------------------
1. Go to https://dash.cloudflare.com -> Storage & Databases -> KV
2. Click into "marmees-reviews"
3. Each entry is one review: name, star rating, review text,
   and when it was submitted.
4. Copy the ones you want live and send them to Claude to add
   to the site, same as always.

------------------------------------------------------------
  IF SUBMISSION ISN'T CONNECTED YET
------------------------------------------------------------
Until you paste the Worker URL into leave-a-review.html, the
form will show an error saying it isn't connected yet, instead
of submitting.

------------------------------------------------------------
  SPAM PROTECTION
------------------------------------------------------------
The form has a basic honeypot (an invisible field bots fill in
but humans never see/fill). Submissions that fill it in are
silently dropped. This isn't bulletproof, but it stops most
automated spam without adding a CAPTCHA to the page.
